# BookFlow

Aplicação Java JPA com Hibernate para sistema de biblioteca.

## Sobre o projeto

A aplicação é para fins didáticos, não há frontend, o objetivo é exercitar lógica de programação, 
orientação a objetos, conexão com banco e operações utilizando o Hibernate, sendo possivel realizar:

- CRUD de clientes
- CRUD de livros
- Emprestimo de livro
- Devolução de livro com pagamento de multa caso haja
- Renovação de empréstimo com multa, caso haja
- Exclusão lógica de clientes e livros para permanecer histórico

## Tecnologias utilizadas

- Java 11
- Hibernate
- PostgreSQL
- Maven
